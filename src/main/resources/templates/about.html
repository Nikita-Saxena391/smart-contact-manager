Smart Contact Manager

Manage your contacts anytime, anywhere — smartly and securely.

📌 About This Website

Smart Contact Manager is a web application designed to help users store, manage, and organize their contacts efficiently. It provides a secure platform where users can access their contacts from anywhere.

🚀 Features

  • User Registration & Login
  • Add, Update & Delete Contacts
  • Upload Profile Images
  • Secure Authentication
  • Responsive Design

⚙️ Technologies

  • Spring Boot
  • Thymeleaf
  • Bootstrap
  • MySQL
  • Java
